# AI Study Assistant

An AI-powered study assistant built using Python, Streamlit, and OpenAI/OpenRouter API.

## Features
- AI Summary Generator
- Quiz Generator
- Flashcards Generator
- AI Topic Explainer

## Tech Stack
- Python
- Streamlit
- OpenAI / OpenRouter API
- dotenv

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
